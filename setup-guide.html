STEP 1 / 5

LINE公式アカウントを開設する

所要時間：約10〜15分 ／ 最初の1回のみ
1
アカウント作成
  1. LINE Official Account Manager にアクセスします。
  2. 「アカウント開設」をクリックします。
  3. お持ちの個人LINEアカウントでログインします。
  4. 以下の情報を入力してアカウントを作成します。
    ・アカウント名（例：脳力開発ドリル）
    ・業種：教育・学習
    ・メールアドレス
2
Messaging APIを有効化する
  1. 管理画面の左メニューから「設定」→「Messaging API」を選択します。
  2. 「Messaging APIを利用する」をクリックします。
  3. 「LINE Developersコンソール」が開きます。
3
チャネルアクセストークンを発行する
  1. LINE Developersコンソールで、作成したチャネルを選択します。
  2. 「Messaging API設定」タブを開きます。
  3. 一番下の「チャネルアクセストークン（長期）」の「発行」ボタンをクリックします。
  4. 表示されたトークン（長い文字列）を必ずコピーして保存してください。
⚠ 重要： このトークンは後でGASに貼り付けます。他人に見せないよう注意してください。
✅ 完了チェック
STEP 2 / 5

Google Apps Script（GAS）を設定する

所要時間：約10〜15分 ／ 最初の1回のみ
1
スプレッドシートを作成する
  1. Google スプレッドシート を開き、新しいシートを作成します。
  2. シート名は「LINE自動送信」など、わかりやすい名前にします。
2
GASエディタを開く
  1. スプレッドシートのメニューから「拡張機能」→「Apps Script」をクリックします。
  2. GASエディタが開きます。
3
プログラムを貼り付ける
  1. エディタに最初から書いてある function myFunction() {} をすべて削除します。
  2. 下のコードをコピーして貼り付けます。
  3. 右上の「保存」ボタン（フロッピーディスクアイコン）をクリックします。

function sendDailyMessage() {
  var props = PropertiesService.getScriptProperties();
  var token = props.getProperty('LINE_CHANNEL_ACCESS_TOKEN');
  if (!token) { Logger.log('エラー: LINE_CHANNEL_ACCESS_TOKEN が設定されていません。'); return; }
  var today = new Date();
  var dayOfWeek = today.getDay();
  var dayNames = ['日', '月', '火', '水', '木', '金', '土'];
  var msg = getDayMessage(dayOfWeek);
  var flexMessage = buildFlexMessage(msg, dayNames[dayOfWeek], today);
  sendBroadcast(token, flexMessage);
  Logger.log('送信完了: ' + today.toLocaleString('ja-JP'));
}

function sendBroadcast(token, flexMessage) {
  var options = {
    "method": "post",
    "headers": { "Content-Type": "application/json", "Authorization": "Bearer " + token },
    "payload": JSON.stringify({ "messages": [flexMessage] }),
    "muteHttpExceptions": true
  };
  var response = UrlFetchApp.fetch("https://api.line.me/v2/bot/message/broadcast", options);
  Logger.log('レスポンスコード: ' + response.getResponseCode());
}

よくある質問
メッセージの内容を変更したい場合は？
Code.gs の getDayMessage 関数内の各曜日の noryoku（脳力開発テーマ）と sixhat（シックスハットの問い）のテキストを直接書き換えてください。
送信時刻を変更したい場合は？
GASのトリガー設定で時刻を変更してください。「午前5時〜6時」などに変更することで、より早い時間に送信できます。
特定の人だけに送りたい場合は？
ブロードキャスト（全員送信）ではなく、プッシュメッセージAPIを使用します。その場合は送信先のユーザーIDが必要になります。プログラムの改修が必要ですので、ご要望があればご相談ください。
エラーが出た場合は？
GASエディタの「実行ログ」を確認してください。「LINE_CHANNEL_ACCESS_TOKEN が設定されていません」と表示された場合は、STEP 2-4のスクリプトプロパティの設定を再確認してください。
